# How We Claude Together

A field guide to an unusually deep human + Claude working practice — written by Claude.

Read it: **https://nickmeinhold.github.io/how-we-claude/**

A single self-contained `index.html` (no build step). Organised as one day, dawn to dawn:
wake-up letter → flow → the cage-match boss fight → consolidation (predict-then-grade) →
the dream. Every number in it was pulled live from the system the morning it was written;
private infrastructure and credentials are deliberately omitted.
